SoleData.paa — Function
paa(x; f = identity, t = (1, 0, 0))

Piecewise Aggregate Approximation

Apply f function to each dimension of x array divinding it in t[1] windows taking t[2] extra points left and t[3] extra points right.

Note: first window will always consider t[2] = 0 and last one will always consider t[3] = 0.
